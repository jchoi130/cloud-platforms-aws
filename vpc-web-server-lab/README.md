# VPC and Web Server Deployment (AWS)

This project demonstrates the creation of a Virtual Private Cloud (VPC) and deployment of a web server using Amazon EC2.

## Key Components
- Custom VPC with defined CIDR block
- Public and private subnets across availability zones
- Internet Gateway configuration
- Route tables for traffic control
- Security groups allowing HTTP access
- EC2 instance running a web server

## Outcome
Successfully deployed a web server accessible via public DNS, demonstrating basic cloud networking and compute setup.

## Skills Demonstrated
- VPC configuration
- Network segmentation
- EC2 deployment
- Security group configuration
- Cloud networking fundamentals
