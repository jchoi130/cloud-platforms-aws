# Cloud Platforms – AWS Infrastructure Project

This repository showcases practical cloud infrastructure work completed as part of my university studies in Cloud Platforms.

The project demonstrates the setup and configuration of core AWS services used to support a basic cloud environment, including networking, compute, and database components.

## Key Skills Demonstrated
- Virtual Private Cloud (VPC) configuration
- Public and private subnet design
- Internet Gateway and route table configuration
- EC2 instance deployment
- Amazon RDS database setup
- Basic cloud security and connectivity concepts
- Structured documentation of deployment steps

## AWS Services Used
- Amazon VPC
- Amazon EC2
- Amazon RDS
- Route Tables
- Internet Gateway
- Security Groups

## Repository Structure
- /screenshots – AWS console screenshots showing setup and configuration
- README.md – Project overview and explanation

## Project Overview
This project involved creating a basic AWS environment to simulate a small cloud deployment. The environment included a VPC, public and private subnets, routing components, an EC2 instance, and a relational database using Amazon RDS.

The aim was to understand how cloud resources are provisioned and connected securely within a cloud platform.

## Projects

- RDS Database Lab → AWS RDS setup and web app connection  
- VPC Web Server Lab → Virtual network and EC2 deployment  
- SA Brewery Cloud Migration → AWS architecture design (EC2, RDS, VPC)

## Notes
This repository is intended to demonstrate foundational cloud infrastructure skills and practical understanding of AWS services. It does not include credentials, private keys, or sensitive account information.
