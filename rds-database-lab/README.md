# RDS Database Deployment (AWS)

This project demonstrates the deployment and configuration of an Amazon RDS database within a private subnet in AWS.

## Key Components
- DB subnet group across multiple availability zones
- RDS MySQL instance deployment
- Security group allowing database access from the web server only
- Database connectivity through the web application
- Private database design for improved security

## Outcome
Successfully deployed and connected a MySQL database using Amazon RDS, demonstrating foundational cloud database and networking concepts.

## Skills Demonstrated
- Amazon RDS configuration
- Private subnet database deployment
- Security group configuration for database access
- Database connectivity validation
- Cloud database fundamentals
